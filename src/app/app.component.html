Hosted Code Snippets
Overview

This project demonstrates using an Angular Component to render source files from hosted sites like GitHub. This is accomplished by retrieving the raw.githubusercontent.com source file using HTTP.

Hosted Snippet Attributes

file

Name of the file to show in the snippet header

language

Language the source file is written in

maxHeight

Max height the code container should render before enabling scrolling. Default is 100%.

url

Fully qualified location of the raw source

Example

<hosted-snippet file="package.json" language="json" maxHeight="500px" url="https://raw.githubusercontent.com/JaimeStill/hosted-code-snippets/master/package.json"></hosted-snippet>
